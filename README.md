# Jatashankartent--website

This is the official website for **Jata Shankar Tent House**.

## Features
- Public landing page with images
- Contact & WhatsApp integration
- Billing and booking system (linked web app)
- Owner approval workflow
- PWA support for business owner

## Tech Stack
- HTML, CSS
- Vercel / Netlify (Frontend)
- Firebase (Billing & Booking App)
